# gogoigo

囲碁の基本用語（キリ、ツギ、ハネ…など19種類）を、盤面をクリックしながら体験的に学べる、**用語学習のためのWebアプリ**です。

🔗 https://jacqueline37.github.io/gogoigo/

## 特徴

- 19種類の基本用語を、1問ずつ盤面クリックで体験しながら学べる
- 前提知識として囲碁の基本ルール（呼吸点・アタリ・石を取る・コウなど）を学べる「ルール編」を用意
- 日本語 / English の言語切り替えに対応
- スマートフォンなど狭い画面でも操作しやすいレイアウト

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルドや依存パッケージのインストールは不要です。

左の一覧から用語を選び、盤面の交点をクリックして「この手かな？」を試してください。正解すると次の問題へ自動的に進みます。

囲碁のルール（呼吸点・アタリ・石を取る・コウなど）がまだ分からない方は、用語を理解するための前提知識として `rules.html`（ルール編）を先に触ってみてください。あくまで用語編の理解を助けるための補助ページです。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 用語編（本編） |
| `rules.html` | ルール編（前提知識） |
| `app.js` / `stages.js` | 用語編のロジックと問題データ |
| `rules.js` / `rules-stages.js` | ルール編のロジックと問題データ |
| `go-rules.js` | 囲碁のルール判定（呼吸点・石を取る・コウなど）共通ロジック |
| `board-ui.js` | 盤面の描画・クリック処理 |
| `i18n.js` | 日本語 / English の文言と切り替え処理 |
| `styles.css` | 全体のスタイル |
