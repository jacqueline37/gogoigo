# gogoigo

囲碁の基本用語（キリ、ツギ、ハネ…など19種類）を、盤面をクリックしながら体験的に学べるWebアプリです。

## 使い方

ビルド不要の静的サイトです。`index.html` をブラウザで開くだけで動作します。

1. 左の一覧（またはモバイルでは「用語一覧を開く」ボタン）から用語を選ぶ
2. 説明を読み、盤面の交点をクリックして「この手かな？」を試す
3. 正解すると次の問題へ自動的に進む

## ファイル構成

- `index.html` — ページ本体
- `styles.css` — スタイル
- `stages.js` — 19問分の問題データ（盤面の石の配置、正解、説明文など）
- `app.js` — 盤面の描画とクリック判定のロジック
