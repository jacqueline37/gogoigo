# gogoigo

囲碁の基本用語（キリ、ツギ、ハネ…など19種類）を、盤面をクリックしながら体験的に学べるWebアプリです。

## 使い方

`index.html` をブラウザで開くだけで動作します。囲碁が初めての方は `rules.html`（ルール編）から始めるのがおすすめです。

- **ルール編**（`rules.html`）: 呼吸点・アタリ・石を取る・コウなど、囲碁の基本ルールを実際に石を打ちながら体験できます。
- **用語編**（`index.html`）: キリ、ツギ、ハネ…などの基本用語19種類を、盤面をクリックしながら覚えられます。左の一覧から用語を選び、盤面の交点をクリックして「この手かな？」を試してください。正解すると次の問題へ自動的に進みます。
